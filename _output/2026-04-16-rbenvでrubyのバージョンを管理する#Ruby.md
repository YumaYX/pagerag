# 質問
1. rbenvでrubyのバージョンを管理する利点は何ですか？
2. rbenvでrubyのバージョンをインストール・切り替えるにはどうすれば良いですか？
3. rbenvとrvmの違いは何ですか？

# 回答
**1. rbenvでrubyのバージョンを管理する利点は何ですか？**
Macにデフォルトで入っているrubyのバージョンが古い場合に、rbenvを使用して最新バージョンで動作させるための管理ができます。

**2. rbenvでrubyのバージョンをインストール・切り替えるにはどうすれば良いですか？**

*   **利用可能なバージョンの確認:** `$ rbenv install --list` を実行します。
*   **グローバルなバージョン設定（環境全体に適用）:** `$ rbenv global [バージョン]` を実行します。
*   **ローカルなバージョン設定（ディレクトリ内限定）:** `% rbenv local [バージョン]` を実行します。
*   **現在設定されているバージョンと環境の確認:**
    *   `$ rbenv version` で現在のバージョンを確認できます。
    *   `$ rbenv versions` で、インストールされているすべてのバージョンを確認できます。
*   **設定したローカルなバージョンを解除する場合:** `% rbenv shell --unset` を実行します。

**3. rbenvとrvmの違いは何ですか？**
