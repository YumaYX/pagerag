# 質問
1. Windows 11でUEFI（BIOS）画面に入るにはどうすればよいですか？
2. Windows 11の起動時にUEFI（BIOS）画面に入るための具体的な手順を教えてください。
3. Windows 11でUEFI（BIOS）画面に入る際に、キーボードのどのキーを押せば良いですか？

# 回答
**1. Windows 11でUEFI（BIOS）画面に入るにはどうすればよいですか？**

Windows 11のメニューから「オプションの選択」画面を経由して入ることができます。

**【手順】**

1.  まず、スタートメニュー、ログイン画面、またはWindowsの設定画面から「オプションの選択」画面を表示させます。
2.  「オプションの選択」画面が表示されたら、「トラブルシューティング」をクリックします。
3.  「トラブルシューティング」画面が表示されたら、「詳細オプション」をクリックします。
4.  「詳細オプション」画面が表示されたら、「UEFIファームウェアの設定」をクリックします。
5.  「UEFIファームウェアの設定」画面が表示されたら、「再起動」をクリックすると、UEFI（BIOS）設定画面が表示されます。

**【補足：オプションの選択画面の表示方法】**
「オプションの選択」画面を表示させるには、以下の方法が代表的です。（いずれの方法でもキーボードの[Shift]キーが必要です。）

*   **方法1：スタートメニューから**
    スタートメニューの「電源」アイコンをクリックして表示されるメニューから、キーボードの**[Shift]キー**を押しながら「再起動」をクリックします。
*   **方法2：ログイン画面から**
    ログイン画面の右下の電源アイコンをクリックして表示されるメニューから、キーボードの**[Shift]キー**を押しながら「再起動」をクリックします。
*   **方法3：Windowsの設定から**
    Windowsの設定画面を開き、「システム」→「回復」の順に開き、表示される「回復」画面内の「PCの起動をカスタマイズする」の項目にある「今すぐ再起動」をクリックします。

**【注意】**
この操作には、Bluetooth接続ではないキーボードやマウスを使用する必要があります。

***

**2. Windows 11の起動時にUEFI（BIOS）画面に入るための具体的な手順を教えてください。**

（※提供された文脈に記載されているのは「Windows 11上から」の進入方法であり、起動時を直接的に指示する手順は記載されていません。しかし、文脈内の「オプションの選択」画面の表示手順をご案内します。）

**【「オプションの選択」画面を表示させる手順（代表的なもの）】**

1.  **スタートメニューから：**
    スタートメニューの「電源」アイコンから、キーボードの[Shift]キーを押しながら「再起動」をクリックします。
2.  **ログイン画面から：**
    ログイン画面の右下の電源アイコンから、キーボードの[Shift]キーを押しながら「再起動」をクリックします。
3.  **Windowsの設定から：**
    Windowsの設定画面を開き、「システム」→「回復」に進み、表示された「回復」画面内の「PCの起動をカスタマイズする」にある「今すぐ再起動」をクリックします。

**【その後のUEFI（BIOS）画面へ進む手順】**

1.  「オプションの選択」画面が表示されたら、「トラブルシューティング」をクリックします。
2.  「トラブルシューティング」画面が表示されたら、「詳細オプション」をクリックします。
3.  「詳細オプション」画面が表示されたら、「UEFIファームウェアの設定」をクリックします。
4.  「UEFIファームウェアの設定」が表示されたら、「再起動」をクリックします。

***

**3. Windows 11でUEFI（BIOS）画面に入る際に、キーボードのどのキーを押せば良いですか？**

「オプションの選択」画面を表示させる操作を行う際は、**[Shift]キー**を押しながら「再起動」をクリックする必要があります。
